# DigiNotes

A web application for creating and sharing personalized digital letters.

DigiNotes allows users to write a letter, add music, attach photos, customize its appearance, and generate a shareable link. Recipients can open the link and experience the completed letter in a clean, interactive format.

## Features

- Multiple letter themes and color variants
- Handwritten-style font options
- YouTube and Spotify integration
- Photo attachments
- Decorative stickers
- Shareable links
- Responsive design
- Supabase-powered storage

## How It Works

1. Select a theme
2. Add a song (optional)
3. Write your letter
4. Add photos and stickers (optional)
5. Customize the appearance
6. Generate a shareable link
7. Share it with anyone

## Tech Stack

### Frontend

- HTML5
- CSS3
- Vanilla JavaScript

### Backend

- Supabase

### Integrations

- YouTube IFrame API
- Spotify Embed Player
- Google Fonts
